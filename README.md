# Local Storage Project

## Description

A simple web application built using HTML, CSS, and JavaScript that demonstrates how to use Local Storage in the browser. Users can store, retrieve, and delete data using a simple interface.

## Features

* Add data to Local Storage
* Search data using a key
* Delete all stored data
* Responsive and attractive user interface
* Pure HTML, CSS, and JavaScript

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Browser Local Storage API

## Project Structure

Local-Storage-Project/

├── index.html

├── style.css

├── script.js

└── README.md

## How to Run

1. Download or clone the repository.
2. Open `index.html` in any web browser.
3. Start adding, searching, and deleting data from Local Storage.

## Learning Outcomes

* Understanding Local Storage API
* DOM Manipulation
* Event Handling in JavaScript
* Separating HTML, CSS, and JavaScript files
